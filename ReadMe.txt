# README - Projeto de Busca em Grid com Python e Flask

## Descrição do Projeto
Este projeto implementa um algoritmo de busca em grid (matriz) utilizando Python como backend e Flask como framework web. A aplicação permite visualizar o processo de busca em um grid interativo, onde é possível definir pontos de início, fim e obstáculos.

## Pré-requisitos
Antes de executar o projeto, certifique-se de ter instalado:
- Python 3.6 ou superior
- pip (gerenciador de pacotes do Python)

## Instalação
1.Crie um ambiente virtual (recomendado):
python -m venv venv


2. Ative o ambiente virtual:
   - No Windows:
     ```
     .\venv\Scripts\activate
     ```
   - No Linux/MacOS:
     ```
     source venv/bin/activate
     ```
## Executando o Projeto

1. Com o ambiente virtual ativado, execute o servidor Flask:
python app.py 

2. Acesse a aplicação no seu navegador:
http://localhost:5000
